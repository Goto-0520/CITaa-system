# CITaa-system

体育会自治会の業務を効率化するシステム

## 機能

- 📋 出欠確認：フォーム回答と名簿を照合
- 📄 名簿PDF生成：団体別にパスワード付きPDFを生成
- 💰 会計集計：収支の自動集計
- 📷 書類スキャン：OCRでExcelに自動入力

## 必要環境

- Python 3.13
- Google アカウント
- Google Colab
